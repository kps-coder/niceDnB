Guide

  • * 레이아웃 : layout.html 참조
  • * 구현방식 : 반응형 웹
  • * 해상도

    - PC : 1920px기준 (최소 해상도 1366px)

    - Mobile: 360px ~ 720px / 비율16:9 비율(Galaxy, iPhone)

  • * 브라우저 호환 : IE11, Edge, chrome, Safari, Firefox, Opera
  • * 문서 유형 : HTML5, CSS3
  • * 인코딩 방식 : UTF-8
  • * 웹접근성 : 웹접근성 준수 목표
  • * 장차법 : 한국정보화진흥원에서 제시하는 22개 웹접근성 항목을 기준으로 구현

Css Guide

  • * 파일위치 : assets > css
  • * main.css : 메인 전용
  • * default.css : 초기화, 레이아웃 정의
  • * common.css : 팝업, form, 버튼 그외 content

images Guide

  • * 파일위치 : assets > images
  • * 이미지가 많지 않아 이미지 폴더 하나 사용

js Guide

  • * 파일위치 : assets > js
  • * lib 폴더 : 라이브러리 전용 폴더

    fullpage.js : main 전용

    jquery-3.7.0.min.js

    jquery-ui.min.js : datepicker 사용

    swiper.min.js : 주로 main 사용

  • *ui_common.js : 퍼블 전용 / GNB, TAB, sitemap, accordion, tooltip, 레이어 팝업

admin Guide

  • * 파일위치 : admin 전용 폴더
  • * front와 별도 관리이지만 폴더 구조 및 css는 front와 유사
  • * 레이아웃 구조 : admin > html > layout.html 참조
